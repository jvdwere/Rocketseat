Funcoes construtoras sao utilizadas para retornar um object em uma variavel, e possuem
 uma estrutura propria.

    Exemplo 1

        function User(){
            this
        }
        const joao = new User

    A principio tudo normal ne? Aqui podemos ver um padrao da comunidade, Funcoes
     construtoras sao declaradas com a primeira letra em maiusculo User() podendo
     ser qqr nome q faca sentido em seu uso. Na variavel joao é usado uma exprecao,
     new, e é exatame isso q torna a funcao User() uma funcao construtora, e ela vai
     criar um objeto e salvar na variavel joao. Dentro da function, temos uma palavra
     chave chamda this (linha 7) esse this, dentro da function vai se referir a 
     variavel joao, podendo assim ser moldado a estrutura do object


    Exemplo 2 

        function User(nome, idade){
            this.nome = nome
            this.idade = idade
        }
        const joao = new User("Joao Victor",24)
        console.log(joao)
    
    Podemos tbm usar function em nosso object

    Exemplo 3 

        function User(nome, idade){
            this.nome = nome
            this.idade = idade
            this.work = function() {
                return `O ${nome} tem ${idade} anos, e esta estudando JavaScript`
        }
        }
        const joao = new User("Joao Victor",24)
        console.log(joao.work())
